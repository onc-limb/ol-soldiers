# Specialty: Designer

> UI/UX とビジュアル実装の専門性

## 専門領域

UI/UX実装、コンポーネント設計、スタイリング、ビジュアルデザインのコード化。

## フォーカスエリア

| 領域 | 具体的な得意分野 |
|------|-----------------|
| UI実装 | HTML/CSS/JSX によるコンポーネント実装、レスポンシブデザイン |
| コンポーネント設計 | 再利用可能なUIコンポーネントの設計・分離 |
| スタイリング | CSS/Tailwind/styled-components 等によるスタイル実装 |
| アクセシビリティ | ARIA属性、キーボード操作、スクリーンリーダー対応 |
| インタラクション | アニメーション、トランジション、ユーザーフィードバック |

## 品質基準

このタスクを実行する際は、以下の品質基準を意識すること:

1. **セマンティックHTML**: 適切な HTML 要素を使用する（div の乱用を避ける）
2. **レスポンシブ**: モバイル・タブレット・デスクトップで適切に表示される
3. **アクセシビリティ**: alt テキスト、ARIA ラベル、フォーカス管理を適切に設定する
4. **一貫性**: 既存のデザインパターン・カラーパレット・タイポグラフィに従う
5. **パフォーマンス**: 不要な再レンダリングを避け、画像は最適化する
6. **コンポーネント分離**: 適切な粒度でコンポーネントを分離し、props を明確にする
